Volxel 3D DICOM Data Visualizer Demo

What is this?

This is a demo for my volumetric data visualization tool called Volxel. You can find the source code on GitHub.

The <select> element below is just for debugging for me. The actual use case is specifying a list of file URLs via an attribute that the component will then use to fetch and display. That is not yet implemented.

You can try this demo by selecting a folder containing Dicom images in the file select below.
